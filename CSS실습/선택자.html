태그 선택자

태그 선택자는 태그명과 일치하는 태그를 모두 선택합니다.

따라서 모든 p태그의 색상은 파란색이 됩니다.

아이디 선택자

아이디 선택자는 id 속성값을 이용해서 선택자를 지정합니다.

클래스 선택자

클래스 선택자는 class 속성값을 이용해서 선택자를 지정합니다.

class 속성은 id 속성과는 다르게 속성값을 중복해서 사용할 수 있다.
